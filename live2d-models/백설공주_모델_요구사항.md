# 백설공주 Live2D 모델 제작 요구사항

## 개요
이 문서는 Live2D 모델 "shizuku"를 참고하여 백설공주 캐릭터용 Live2D 모델을 제작하기 위해 필요한 모든 요소를 정리한 것입니다.

---

## 1. 텍스처 이미지 (Texture Images)

### 1.1 텍스처 구성
Live2D 모델은 여러 레이어로 분리된 텍스처 이미지가 필요합니다. 각 텍스처는 **1024x1024 픽셀** 해상도로 제작되어야 합니다.

**필요한 텍스처 파일 (총 5개):**
- `texture_00.png` - 기본 레이어 (배경, 코어 파츠)
- `texture_01.png` - 캐릭터 기본 레이어 (얼굴, 몸통 기본)
- `texture_02.png` - 캐릭터 상세 레이어 (의상, 액세서리)
- `texture_03.png` - 캐릭터 추가 레이어 (손, 팔 등)
- `texture_04.png` - 효과/표정 레이어 (눈, 입, 표정 변화)

### 1.2 백설공주 특화 디자인 요소
- **의상**: 파란색과 노란색이 주조색인 공주 드레스
- **머리**: 검은 머리카락 (단발 또는 긴 머리)
- **악세서리**: 왕관 또는 리본
- **색상 팔레트**: 흰색 피부, 검은 머리, 파란/노란 드레스

---

## 2. 파츠 구성 (Parts Structure)

Live2D 모델은 각 부위가 독립적인 파츠로 구성되어야 합니다. 다음 파츠들이 필요합니다:

### 2.1 얼굴 파츠
- **얼굴 기본** (`PARTS_01_FACE_001`) - 얼굴 기본형
- **눈** (`PARTS_01_EYE_001`) - 눈 기본형
- **눈알** (`PARTS_01_EYE_BALL_001`) - 눈동자
- **눈알 효과** (`PARTS_01_EYE_BALL_002`, `PARTS_01_EYE_BALL_003`) - 눈 깜빡임, 반짝임 효과
- **눈썹** (`PARTS_01_BROW_001`) - 눈썹
- **코** (`PARTS_01_NOSE_001`) - 코
- **입** (`PARTS_01_MOUTH_001`) - 입
- **볼** (`PARTS_01_HOHO_01`) - 볼 (홍조 효과용)
- **얼굴 그림자** (`PARTS_01_FACESHADOW_001`) - 얼굴 그림자

### 2.2 머리카락 파츠
- **앞머리** (`PARTS_01_HAIR_FRONT_001`) - 앞머리 (물리 효과 적용)
- **옆머리** (`PARTS_01_HAIR_SIDE_001`) - 옆머리 (물리 효과 적용)
- **뒤머리** (`PARTS_01_HAIR_BACK_001` 또는 `PARTS_01_HAIR_BACK_002`) - 뒤머리 (물리 효과 적용)
- **트윈테일** (선택사항, `PARTS_01_HAIR_TWIN_L`, `PARTS_01_HAIR_TWIN_R`) - 양쪽 포니테일

### 2.3 몸체 파츠
- **몸통** (`PARTS_01_BODY`) - 몸통 기본
- **목** (`PARTS_01_NECK`) - 목
- **의상** (`PARTS_01_UNIFORM` 또는 커스텀 의상 파츠) - 드레스
- **왼팔** (`PARTS_01_ARM_L_01`, `PARTS_01_ARM_L_02`) - 왼팔 상/하단
- **오른팔** (`PARTS_01_ARM_R_01`, `PARTS_01_ARM_R_02`) - 오른팔 상/하단
- **왼손** (`PARTS_01_HAND_L`) - 왼손
- **오른손** (`PARTS_01_HAND_R`) - 오른손

### 2.4 기타 파츠
- **배경** (`PARTS_01_BACKGROUND`) - 배경 (선택사항)
- **효과 파츠** (`PARTS_01_KIRAKIRA`) - 반짝임 효과 (선택사항)
- **코어 파츠** (`PARTS_CORE`) - 중심 좌표계

## 3. 애니메이션 파라미터 (Animation Parameters)

Live2D 모델은 다양한 파라미터를 통해 움직임을 제어합니다. 다음 파라미터들이 필요합니다:

### 3.1 얼굴 회전 파라미터
- `PARAM_ANGLE_X` - 얼굴 좌우 회전 (X축)
- `PARAM_ANGLE_Y` - 얼굴 상하 회전 (Y축)
- `PARAM_ANGLE_Z` - 얼굴 좌우 기울임 (Z축)

### 3.2 눈 파라미터
- `PARAM_EYE_L_OPEN` - 왼쪽 눈 깜빡임 (0.0: 닫힘, 1.0: 열림)
- `PARAM_EYE_R_OPEN` - 오른쪽 눈 깜빡임 (0.0: 닫힘, 1.0: 열림)
- `PARAM_EYE_BALL_X` - 눈동자 좌우 이동
- `PARAM_EYE_BALL_Y` - 눈동자 상하 이동
- `PARAM_EYE_BALL_FORM` - 눈동자 변형
- `PARAM_EYE_BALL_KIRAKIRA` - 눈 반짝임 효과

### 3.3 눈썹 파라미터
- `PARAM_BROW_L_Y` - 왼쪽 눈썹 상하 이동
- `PARAM_BROW_R_Y` - 오른쪽 눈썹 상하 이동
- `PARAM_BROW_L_X` - 왼쪽 눈썹 좌우 이동
- `PARAM_BROW_R_X` - 오른쪽 눈썹 좌우 이동
- `PARAM_BROW_L_ANGLE` - 왼쪽 눈썹 각도
- `PARAM_BROW_R_ANGLE` - 오른쪽 눈썹 각도
- `PARAM_BROW_L_FORM` - 왼쪽 눈썹 변형
- `PARAM_BROW_R_FORM` - 오른쪽 눈썹 변형

### 3.4 입 파라미터
- `PARAM_MOUTH_OPEN_Y` - 입 벌림 정도 (0.0: 닫힘, 1.0: 최대 벌림) - **LipSync에 사용**
- `PARAM_MOUTH_FORM` - 입 모양 변형
- `PARAM_MOUTH_SIZE` - 입 크기 조절

### 3.5 표정 파라미터
- `PARAM_TERE` - 수줍음/홍조 효과

### 3.6 몸체 파라미터
- `PARAM_BODY_X` - 몸통 X축 회전
- `PARAM_BODY_Y` - 몸통 Y축 회전
- `PARAM_BODY_Z` - 몸통 Z축 회전
- `PARAM_BREATH` - 호흡 애니메이션

### 3.7 팔/손 파라미터
- `PARAM_ARM_L` - 왼팔 기본 회전
- `PARAM_ARM_L_02` - 왼팔 추가 회전
- `PARAM_HAND_L` - 왼손 회전
- `PARAM_ARM_R` - 오른팔 기본 회전
- `PARAM_ARM_R_02` - 오른팔 추가 회전
- `PARAM_HAND_R` - 오른손 회전

### 3.8 머리카락 물리 효과 파라미터
- `PARAM_KAMIYURE_FRONT` - 앞머리 흔들림
- `PARAM_KAMIYURE_BACK` - 뒤머리 흔들림
- `PARAM_KAMIYURE_SIDE_L` - 왼쪽 옆머리 흔들림
- `PARAM_KAMIYURE_SIDE_R` - 오른쪽 옆머리 흔들림
- `PARAM_KAMIYURE_TWIN_L` - 왼쪽 트윈테일 흔들림
- `PARAM_KAMIYURE_TWIN_R` - 오른쪽 트윈테일 흔들림

## 4. 물리 설정 (Physics Settings)

머리카락과 의상의 자연스러운 움직임을 위해 물리 효과가 필요합니다:

### 4.1 물리 효과 그룹
- **앞머리 흔들림** (`PhysicsSetting1`) - 얼굴 회전에 따라 앞머리가 자연스럽게 흔들림
- **옆머리 흔들림** (`PhysicsSetting2`) - 얼굴 회전에 따라 옆머리가 자연스럽게 흔들림
- **트윈테일 흔들림** (`PhysicsSetting3`) - 트윈테일이 있는 경우 적용

### 4.2 물리 효과 입력
- 얼굴 회전 각도 (`PARAM_ANGLE_X`, `PARAM_ANGLE_Y`, `PARAM_ANGLE_Z`)를 입력으로 사용
- 중력 설정: Y축 -1 방향

---

## 5. 포즈 설정 (Pose Settings)

팔의 자연스러운 움직임을 위한 포즈 그룹:

- **왼팔 그룹**: 왼팔 상/하단이 함께 움직이도록 연결
- **오른팔 그룹**: 오른팔 상/하단이 함께 움직이도록 연결

---

## 6. 모션 파일 (Motion Files)

기본 모션 4종이 필요합니다:

- **FlickUp** (`01.motion3.json`) - 위로 스와이프 모션
- **Tap** (`02.motion3.json`) - 탭 모션
- **Flick3** (`03.motion3.json`) - 스와이프 모션
- **Idle** (`04.motion3.json`) - 대기 모션

---

## 7. 이미지 생성 AI에게 요청 가능한 부분 vs 추가 작업 필요 부분

### 7.1 ✅ 이미지 생성 AI에게 요청 가능한 부분

**이미지 생성 AI는 다음 작업을 수행할 수 있습니다:**

1. **텍스처 이미지 생성** (5개 파일)
   - 백설공주 캐릭터 일러스트 제작
   - 각 레이어별로 분리된 이미지 생성
   - 투명 배경 PNG 파일 생성

2. **파츠 분리 이미지 생성**
   - 각 부위(얼굴, 머리카락, 의상 등)를 독립적으로 그린 이미지
   - 여러 상태의 눈, 입 이미지 생성

**⚠️ 제한사항:**
- 이미지 생성 AI는 **일러스트만 생성**할 수 있습니다
- Live2D 리깅을 위한 정확한 파츠 분리는 AI가 완벽하게 하기 어려울 수 있습니다
- 생성된 이미지를 Live2D Cubism Editor에서 수동으로 편집/분리해야 할 수 있습니다

### 7.2 ❌ 이미지 생성 AI가 할 수 없는 부분 (추가 작업 필요)

다음 작업들은 **Live2D Cubism Editor**에서 수동으로 수행해야 합니다:

1. **리깅 (Rigging)** - 각 파츠에 본(Bone) 배치 및 변형 설정
2. **파라미터 설정** - 애니메이션 파라미터 정의 및 연결
3. **물리 효과 설정** - 머리카락 흔들림 등 물리 시뮬레이션
4. **포즈 설정** - 팔 움직임 그룹 설정
5. **모션 제작** - 기본 모션 4종 제작
6. **모델 내보내기** - `.moc3`, `.model3.json` 등 런타임 파일 생성

## 8. 이미지 생성 AI 요청 시 프롬프트 가이드

### 8.1 기본 요구사항
```
캐릭터: 백설공주 (Snow White)
스타일: Live2D 모델용 2D 일러스트, 애니메이션 스타일
해상도: 각 텍스처 1024x1024 픽셀
레이어 수: 5개 텍스처 파일
형식: PNG, 투명 배경
```

### 8.2 디자인 가이드라인

**캐릭터 외형:**
- 피부: 매우 흰색, 깨끗한 피부톤
- 머리: 검은색, 긴 머리 또는 단발
- 눈: 큰 눈, 갈색 또는 검은색 눈동자
- 의상: 파란색과 노란색이 주조색인 공주 드레스
- 악세서리: 작은 왕관 또는 리본

**레이어 분리 가이드:**
1. **texture_00.png**: 배경, 코어 파츠
2. **texture_01.png**: 얼굴 기본, 몸통 기본, 머리카락 기본
3. **texture_02.png**: 의상 상세, 악세서리, 머리카락 디테일
4. **texture_03.png**: 팔, 손, 손가락
5. **texture_04.png**: 눈 (여러 상태), 입 (여러 상태), 표정 변화 요소

**중요:**
- 각 파츠는 독립적으로 분리 가능해야 함
- 투명 배경 (PNG 알파 채널)
- 각 파츠의 경계가 명확해야 함
- Live2D 리깅을 위해 파츠들이 적절히 분리되어 있어야 함

### 8.3 파츠별 상세 요구사항

**얼굴:**
- 얼굴 기본형 (얼굴 윤곽)
- 눈 (열림/닫힘 상태별로 분리)
- 눈동자 (별도 레이어)
- 눈썹 (위치 조정 가능하도록)
- 코 (작고 자연스러운)
- 입 (여러 모양: 닫힘, 벌림, 웃음 등)
- 볼 (홍조 효과용)

**머리카락:**
- 앞머리 (물리 효과 적용 가능하도록 분리)
- 옆머리 (좌우 분리)
- 뒤머리 (또는 트윈테일)
- 각 부분이 독립적으로 움직일 수 있어야 함

**몸체:**
- 목
- 몸통 기본형
- 의상 (드레스)
- 팔 (상/하단 분리)
- 손 (손가락 디테일)

### 8.4 이미지 생성 AI 프롬프트 예시

**텍스처 01 (기본 레이어) 요청 예시:**
```
백설공주 캐릭터, 정면 뷰, 2D 애니메이션 스타일
- 매우 흰색 피부, 검은색 긴 머리
- 큰 눈, 갈색 눈동자
- 파란색과 노란색 공주 드레스
- 작은 왕관
- 투명 배경, 1024x1024 픽셀
- Live2D 모델용, 파츠 분리 가능한 구조
```

**텍스처 04 (표정 레이어) 요청 예시:**
```
백설공주 캐릭터의 눈과 입 파츠
- 눈: 열림 상태, 반쯤 감김 상태, 닫힘 상태 각각
- 입: 닫힘, 벌림, 웃음 상태 각각
- 투명 배경, 1024x1024 픽셀
- Live2D 애니메이션용
```

## 9. Live2D Cubism Editor 작업 필요 사항

3. **물리 효과 설정**: 머리카락 물리 효과 적용
4. **포즈 설정**: 팔 움직임 그룹 설정
5. **모션 제작**: 기본 모션 4종 제작
6. **모델 내보내기**: `.moc3`, `.model3.json` 등 런타임 파일 생성
